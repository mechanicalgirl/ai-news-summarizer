### Agent Safehouse – macOS-native sandboxing for local agents - (Hacker News)

https://agent-safehouse.dev/

# Summary

Agent Safehouse is a lightweight macOS security tool that sandboxes AI agents (like Claude, Codex, and Gemini) to prevent accidental or malicious damage to your system. Using kernel-level sandboxing, it denies file access outside your project directory by default, only granting explicit permissions to your current workspace and toolchains while blocking access to sensitive files like SSH keys and other repositories.

The tool is simple to set up—just download a single Bash script and run agents inside it—with no dependencies or build steps required. Users can customize access permissions, test the sandbox's effectiveness, and even configure shell functions to automatically sandbox agents by default, with the option to bypass sandboxing when needed.

### PCB devboard the size of a USB-C plug - (Hacker News)

https://github.com/Dieu-de-l-elec/AngstromIO-devboard

# Summary

AngstromIO is an open-source project featuring one of the smallest development boards available, measuring just 8.9mm × 9mm and barely longer than a USB-C connector. Based on the ATtiny1616 microcontroller, it includes 2 RGB addressable LEDs, broken-out GPIO and I2C pins, and runs at 5V. The project also includes a dual CH340E programmer board that enables both UPDI programming and serial debugging through separate USB-C connections, plus an on-board 3.3V LDO regulator.

Additionally, the project contains a breadboard-friendly experimentation board for the CH32V003 RISC-V microcontroller, featuring a 4×5 charlieplexed LED matrix and designed for learning purposes. All three PCB designs—the AngstromIO devboard, programmer, and CH32 board—are panelized into a single PCB fabricated in EasyEDA Pro with a purple soldermask. The AngstromIO is Arduino-compatible and works with specialized libraries, while the CH32 board is programmed using MounRiver Studio IDE.

### We should revisit literate programming in the agent era - (Hacker News)

https://silly.business/blog/we-should-revisit-literate-programming-in-the-agent-era/

# Summary

Literate programming—intermingling code with prose narrative to make codebases readable like stories—has historically struggled with adoption because maintaining two parallel narratives (code and explanation) becomes burdensome. The author argues this fundamental problem is solved by AI coding agents, which excel at translation and summarization. Agents can automatically handle the tedious work of keeping prose explanations synchronized with code changes, managing code extraction ("tangling"), and reformatting documentation—tasks that previously made literate programming impractical for larger projects.

The author proposes using AI agents with Org Mode to create executable runbooks and documented processes where code blocks and explanatory text coexist, with results embedded directly in the document. This approach leverages what language models do best while making literate programming finally viable at scale. The author suggests this pattern could improve code quality and reading comprehension, though they acknowledge the Org format's limitation to Emacs and express hope for a more universal solution with similar capabilities. The compelling question remains: can AI agents make large, narrative-style codebases practical?

### FrameBook - (Hacker News)

https://fb.edoo.gg

# FrameBook: A Modern MacBook Retrofit Project

This article documents the author's ambitious project to retrofit a 2006 MacBook with modern internals, specifically a Framework Laptop 13 mainboard with an Intel i7-1280P processor. Starting with beat-up black polycarbonate MacBooks from eBay, the author disassembled them down to the chassis and replaced nearly all internal components while preserving the classic design. The process involved soldering the original keyboard and trackpad circuitry to USB, 3D printing custom standoffs and I/O shields, and carefully integrating modern USB hubs to replace the old ports.

The project took approximately three months and presented numerous challenges, from soldering for the first time to designing custom mounting solutions and recreating the iconic glowing Apple logo using a custom LED panel. The author relied on creative solutions—including superglue, aluminum tape, and 3D-printed components—to make everything fit together. Despite some non-ideal construction methods and lack of documentation along the way, the result is a fully functional "FrameBook" that successfully blends the nostalgic design of the early 2000s MacBook with contemporary performance and features.

### Show HN: Mcp2cli – One CLI for every API, 96-99% fewer tokens than native MCP - (Hacker News)

https://github.com/knowsuchagency/mcp2cli

# Summary

**mcp2cli** is a command-line tool that converts MCP (Model Context Protocol) servers and OpenAPI specs into CLI interfaces at runtime, without requiring code generation. It addresses the "tool sprawl" problem where feeding LLMs large tool schemas wastes significant context tokens on every message—a 50-endpoint API alone can consume 3,579 tokens before conversation even begins.

The tool saves 96-99% of tokens wasted on tool schemas by letting LLMs discover and invoke tools on-demand through human-readable `--list` and `--help` commands (~16 tokens per tool vs. ~121 for native schemas). It supports both MCP servers (HTTP/SSE and stdio modes) and OpenAPI specs (JSON/YAML), includes caching with configurable TTL, and works provider-agnostic across any LLM—Claude, GPT, Gemini, or local models.

### The death of social media is the renaissance of RSS (2025) - (Hacker News)

https://www.smartlab.at/rss-revival-life-after-social-media/

# Summary

Social media is dying due to an overwhelming flood of AI-generated content that drowns out authentic human voices. Generative AI tools have made content creation nearly free and instantaneous, resulting in oversaturated feeds filled with repetitive, shallow material. Combined with algorithmic amplification that prioritizes engagement over truth, social platforms have become less trustworthy and valuable, accelerating a decline that began years ago with advertising overload and bot armies.

In response, RSS (Really Simple Syndication)—an old web technology from the early 2000s—is experiencing a renaissance as the solution to algorithmic control and AI-generated noise. RSS allows users to subscribe directly to chosen sources without algorithmic intermediaries, creating curated, personalized feeds of authentic content. By returning control to users and eliminating corporate algorithms and ads, RSS represents a return to a decentralized internet that prioritizes transparency, authenticity, and individual choice over engagement metrics and data mining.
