### macOS Container Machines - (Hacker News)

https://github.com/apple/container/blob/main/docs/container-machine.md

# Summary

Container machine is a tool from Apple that provides an integrated Linux environment on macOS. It creates lightweight, persistent Linux containers based on standard OCI images with automatic user and home directory sharing. This allows developers to edit code on macOS while building and running applications inside a Linux environment, with seamless access to repositories, dotfiles, and the ability to run real Linux services like databases using systemd.

The tool supports creating multiple container machines for different Linux distributions, resizing resources, and running custom container images. It can be managed through simple commands like `container machine create`, `run`, `stop`, and `rm`, and allows developers to use native macOS tools to inspect artifacts built in the Linux environment without any copy step required.

### Upcoming breaking changes for npm v12 - (Hacker News)

https://github.blog/changelog/2026-06-09-upcoming-breaking-changes-for-npm-v12/

# Summary

npm v12, scheduled for release in July 2026, introduces three major security-focused breaking changes to the default behavior of `npm install`. These changes require users to explicitly opt-in to previously automatic behaviors: executing installation scripts (allowScripts), resolving Git dependencies (--allow-git), and resolving remote URL dependencies (--allow-remote). All changes are currently available as warnings in npm 11.16.0+, allowing developers to prepare in advance.

To prepare, users should upgrade to npm 11.16.0 or later and review the warnings during their normal install process. They can use `npm approve-scripts` to audit and explicitly allow trusted packages' scripts, then commit the resulting allowlist to package.json. After upgrading to v12, only approved scripts will execute, while unapproved ones will be blocked, improving supply chain security by preventing unvetted code execution.

### German ruling declares Google liable for false answers in AI Overviews - (Hacker News)

# Summary

A German regional court has ruled that Google is directly liable for false statements in its AI-generated search overviews, rejecting the company's previous legal protections as a search engine operator. The court classified AI overviews as Google's own content rather than neutral search results, since Google rewrites and judges information "in its own words." In this case, Google's AI had falsely linked two publishers to scams and shady business practices by mixing up information that didn't appear in the linked sources.

The court rejected Google's defense that users could fact-check the results themselves, drawing parallels to press liability laws and noting that users rarely click on source links. The ruling carries significant implications beyond this case: with a 91% accuracy rate, Google's AI Overviews still produce millions of wrong answers daily, potentially creating widespread liability issues for Google and other AI companies like ChatGPT and Claude that paraphrase web content.

### Rich Sutton on AI creativity and discovery - (Hacker News)

https://twitter.com/RichardSSutton/status/2061216087744946656

# Summary

Richard Sutton argues that generative AI trained through supervised learning cannot achieve true novelty and quality simultaneously—it can produce novel outputs through randomness or high-quality outputs by mimicking training data, but not both at once. While this limitation doesn't matter for practical applications like summarization (where novelty is undesirable "hallucination"), it is devastating for science and mathematics, which require genuine creativity and discovery.

Sutton identifies "Discovery" as the missing ingredient in generative AI: a three-step process of variation, evaluation, and selective retention seen in evolution, the scientific method, and reinforcement learning. Generative AI lacks the evaluation step needed to recognize and retain valuable novel outputs, so while it can generate variations, without assessment there is no true discovery. True creative AI systems like AlphaGo and AlphaFold incorporate this missing mechanism through clear objectives or human evaluation that allows them to combine novelty with quality.

### Ultrafast machine learning on FPGAs via Kolmogorov-Arnold Networks - (Hacker News)

https://aarushgupta.io/posts/kan-fpga/

# Summary

This article describes using Kolmogorov-Arnold Networks (KANs) on FPGAs to achieve ultrafast machine learning inference and online learning. The author explains that FPGAs—reconfigurable digital logic devices—are better suited than GPUs for applications requiring ultra-low latency and high hardware efficiency. The key innovation is representing KAN activation functions as lookup tables (LUTs), the fundamental building blocks of FPGAs, enabling direct implementation of neural networks as digital circuits rather than sequential processor instructions.

The approach leverages KANs' natural structure of univariate learnable activation functions, which avoid the exponential scaling problems of multivariate lookup tables used in other LUT-based neural networks. The process involves training KANs in software (PyTorch), then deploying the fixed models on FPGAs by converting each activation function into a lookup table using fixed-point quantization. This architecture achieves significant improvements in speed and hardware efficiency compared to traditional GPU-based machine learning, particularly for applications demanding nanosecond-level latencies.

### The oldest surviving animated feature film at 100 - (Hacker News)

https://www.bbc.com/culture/article/20260603-how-a-26-year-old-german-woman-made-the-worlds-oldest-surviving-animated-feature-film

# Summary

Lotte Reiniger, a 26-year-old German director, created *The Adventures of Prince Achmed* in 1926, which is recognized as the world's oldest surviving animated feature film—predating Walt Disney's *Snow White and the Seven Dwarfs* by over a decade. Using an innovative stop-motion technique with hand-cut silhouette puppets lit from below, Reiniger single-handedly directed, designed characters, and devised the story for this ambitious three-year project that combined Middle Eastern fairy tales into one fantastical narrative.

Though groundbreaking in both technique and achievement—including pioneering an early multiplane camera to create depth—Reiniger's film has been largely overshadowed by Disney's superior marketing and self-promotion. While some historians debate whether an earlier Argentinian film should hold the title (as all copies have been lost), *The Adventures of Prince Achmed* remains a timeless classic that continues to captivate audiences a century later, though it was initially distributed as an experimental rather than entertainment film, limiting its commercial success.
