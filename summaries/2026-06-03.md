### 1-Click GitHub Token Stealing via a VSCode Bug - (Hacker News)

https://blog.ammaraskar.com/github-token-stealing/

# Summary

A critical vulnerability in VSCode's web version (github.dev) allows attackers to steal GitHub OAuth tokens with a single click. The bug exploits VSCode's webview security model, which uses iframes to sandbox untrusted content like Markdown previews. To enable user-friendly features like keyboard shortcuts, VSCode's webviews can send keyboard events to the main window via a message-passing API. However, malicious JavaScript in the webview can forge these keyboard events to simulate user input, allowing attackers to trigger VSCode commands without genuine user interaction. By abusing this mechanism, an attacker can trigger a series of commands that installs a malicious extension through a trusted workspace's local extension folder, ultimately gaining code execution and stealing the user's GitHub token with full repository access.

The attack chain exploits VSCode's extension contribution system, allowing the attacker's malicious extension to define custom keybindings that bypass the publisher trust verification system. Since github.dev is always treated as a trusted workspace, local extensions skip security checks, making it possible to execute arbitrary code and exfiltrate sensitive credentials in just one click.

### Use your Nvidia GPU's VRAM as swap space on Linux - (Hacker News)

https://github.com/c0dejedi/nbd-vram

# Summary

**nbd-vram** is a Linux tool that uses an NVIDIA GPU's unused VRAM as high-priority swap space, designed for laptops with soldered memory and no upgrade options. It works by running a daemon that allocates VRAM through the CUDA driver API and exposes it as a block device via the NBD (Network Block Device) protocol, which the kernel then uses as swap space. This approach avoids limitations of NVIDIA's P2P API (which doesn't work on consumer GPUs) by using standard CUDA memory copy operations instead.

The tool offers significant practical advantages for sporadic memory access patterns common on laptops, delivering 27x faster average latency than NVMe swap (335 us vs. 9.05 ms) due to VRAM's lack of power state delays. While it's slower for sustained sequential throughput (1.1-2.3 GB/s vs. 2.7-2.9 GB/s on NVMe), it excels at the individual page faults that typically occur during memory pressure. Installation is straightforward, with automatic power management that disables swap when unplugged to preserve battery life.

### The Unreasonable Redundancy of Nature's Protein Folds - (Hacker News)

https://research.ligo.bio/posts/unreasonable-redundancy-of-natural-protein-folds/

# Summary

Modern deep learning models for biomolecular design, like AlphaFold3, are being improved by scaling training data—converting vast protein sequence databases into predicted 3D structures. However, researchers at Ligo discovered a fundamental problem: while natural protein sequences are extraordinarily diverse, their actual 3D folds are far more redundant than sequence counts suggest. Proteins can be less than 30% identical in sequence yet share the same fold structure, meaning simply folding more sequences doesn't necessarily add proportional structural diversity for training.

To address this, the authors developed data engineering techniques to cluster the protein universe by structural similarity rather than sequence similarity. They found that existing clustering approaches vastly overcounted unique folds—reporting 2.3 million clusters when the true number of reusable structural neighborhoods is likely closer to 25,000. The challenge stems from predicted structures containing disordered regions, long linkers, and multi-domain proteins that make clustering ill-defined, so they developed filtering methods to identify genuine, usable protein domains rather than including noisy structural artifacts.

### Capstone – multi-platform, multi-architecture disassembly framework - (Hacker News)

https://www.capstone-engine.org/

# Summary

Capstone is a lightweight, open-source disassembly framework designed as the ultimate tool for binary analysis and reverse engineering in the security community. It supports multiple architectures (ARM, x86, MIPS, RISC-V, and many others), provides a clean and intuitive API, and is implemented in C with bindings for numerous programming languages. The framework is thread-safe, cross-platform, and particularly suited for malware analysis and embedded systems.

The project has gained significant industry recognition from leading security researchers who praise its solid implementation, free availability, and embeddability. Capstone is actively maintained with regular releases addressing security vulnerabilities, performance improvements, and expanded architecture support, as evidenced by multiple recent updates including versions 6.0.0-Alpha and 5.0.x patches.
