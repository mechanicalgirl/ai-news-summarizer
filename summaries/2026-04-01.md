### Mad Bugs: Vim vs. Emacs vs. Claude - (Hacker News)

https://blog.calif.io/p/mad-bugs-vim-vs-emacs-vs-claude

# Summary

A security researcher used Claude AI to identify remote code execution (RCE) vulnerabilities in both Vim and Emacs by simply prompting the AI to find bugs. Claude successfully discovered a zero-day RCE in Vim that could be triggered by opening a file, which was immediately patched in version 9.2.0272. When the researcher jokingly suggested switching to Emacs, Claude found an RCE vulnerability there as well, though Emacs maintainers declined to address it, attributing it to git instead.

The article positions these discoveries as a significant moment in cybersecurity, comparing the accessibility of AI-assisted bug hunting to the SQL injection era of the early 2000s. The researchers are launching "MAD Bugs: Month of AI-Discovered Bugs" to showcase additional vulnerabilities uncovered by AI throughout April 2026, highlighting both the power and concerning implications of using advanced AI systems for security research.

### Bring Back MiniDV with This Raspberry Pi FireWire Hat - (Hacker News)

https://www.jeffgeerling.com/blog/2026/minidv-with-raspberry-pi-firewire-hat/

# Summary

Jeff Geerling demonstrates how to use a new FireWire HAT with a Raspberry Pi 5 and PiSugar3 battery to create a portable "Memory Recording Unit" (MRU) that can replace tape in older FireWire/DV cameras and archive MiniDV tapes. The setup costs $150-200 and provides 2-4 hours of battery runtime, with recorded files saved to microSD cards or accessible via WiFi/SFTP.

The Firehat HAT features GPIO buttons, an OLED display for status information, and requires enabling FireWire support in the Linux kernel and installing custom software. As an alternative, users can build an "Open MRU" using a standard Mini PCIe adapter with a FireWire controller, though it requires command-line control. The setup should remain functional until at least 2029, providing a practical solution for those left without FireWire support after Apple dropped it from macOS.

### TinyLoRA – Learning to Reason in 13 Parameters - (Hacker News)

https://arxiv.org/abs/2602.04118

# Summary

This paper introduces TinyLoRA, a novel method that enables language models to learn reasoning tasks with an extraordinarily small number of parameters. The researchers demonstrate that the 8B-parameter Qwen2.5 model can achieve 91% accuracy on the GSM8K mathematical reasoning benchmark using only 13 trained parameters (26 bytes). More broadly, they show this approach recovers 90% of typical performance improvements while training 1,000x fewer parameters across challenging benchmarks like AIME, AMC, and MATH500.

A key finding is that this extreme parameter efficiency is only achievable through reinforcement learning (RL); supervised fine-tuning (SFT) requires 100-1,000x larger parameter updates to reach comparable performance. This work challenges conventional assumptions about the minimum complexity needed for reasoning in language models and suggests that RL-based training methods may be fundamentally more parameter-efficient for learning reasoning tasks.

### Show HN: 1-Bit Bonsai, the First Commercially Viable 1-Bit LLMs - (Hacker News)

https://prismml.com/

# PrismML Summary

PrismML has developed ultra-efficient AI models using 1-bit weights to address the limitations of large language models that are too massive for smartphones and unsustainable for datacenters. Their Bonsai series includes three models (8B, 4B, and 1.7B parameters) that achieve extraordinary efficiency gains: the 1-bit Bonsai 8B requires only 1.15GB of memory while delivering 14× smaller footprint, 8× faster performance, and 5× better energy efficiency compared to standard full-precision 8B models, with over 10× greater intelligence density.

The company, backed by Caltech research, is repositioning AI development around "intelligence density"—maximizing capability per unit of memory and energy rather than simply scaling up model size. Their models are optimized for edge computing, robotics, and real-time applications, with the smallest variant (1.7B) achieving 130 tokens per second on an iPhone, making powerful AI viable directly on consumer devices.

### TruffleRuby - (Hacker News)

https://chrisseaton.com/truffleruby/

# Summary

TruffleRuby is a high-performance implementation of the Ruby programming language built on the JVM using Oracle's Graal dynamic compiler and Truffle AST interpreter framework. Initiated as an internship project in 2013, it has evolved into an independent project under GraalVM (now sponsored by Shopify since 2019) and achieves superior peak performance compared to JRuby while maintaining greater simplicity.

The page serves as a comprehensive resource hub documenting TruffleRuby's development through numerous blog posts, research papers published in top programming language conferences (PLDI, OOPSLA, TOPLAS), and video presentations from major tech conferences. The project encompasses significant research contributions including specialized optimizations for data structures, cross-language interoperability, debugging capabilities, and metaprogramming techniques that have advanced the broader field of dynamic language runtime implementation.

### A dot a day keeps the clutter away - (Hacker News)

https://scottlawsonbc.com/post/dot-system

# Summary

Scott Lawson developed a simple yet effective inventory system for his electronics component collection using colored dot stickers. Each time he uses a box of parts, he adds one dot sticker per day (regardless of how many times the box is opened that day), with each color representing a different year. The system requires only clear storage boxes, labels, and inexpensive sticker sheets, and has required no software or complex tracking over four years of use.

After four years of data, the dots revealed surprising insights: the most-used items aren't specialized or "cool" components, but rather cross-cutting essentials like glue, tape, batteries, connectors, fasteners, and basic electronic components that appear in nearly every project. The system also showed that expensive tools he assumed were essential—like his oscilloscope—were rarely used. By visualizing usage patterns on his shelves, Lawson could confidently identify which components to keep, which to consolidate, and which to discard, helping his collection reach a sustainable equilibrium.

### Show HN: CLI to order groceries via reverse-engineered REWE API (Haskell) - (Hacker News)

https://github.com/yannick-cw/korb

# Korb: REWE Delivery CLI Summary

Korb is an unofficial command-line tool written in Haskell that allows users to programmatically interact with REWE's (a German supermarket chain) APIs to search for products, manage shopping baskets, and order groceries for pickup. The tool outputs JSON for programmatic use and is designed to be controlled by AI agents, enabling workflows like adding items via voice commands and having an agent handle weekly grocery ordering based on purchase history and preferences.

The project includes features such as product search by name or barcode, favorite management, basket management, timeslot selection, order checkout, and order history tracking. Notably, it incorporates formal verification using Lean 4 for the suggestion engine, mathematically proving five properties of the recommendation algorithm and validating it against the Haskell implementation through differential random testing. The tool requires REWE mTLS certificates and is available for macOS and Linux.

### MiniStack (replacement for LocalStack) - (Hacker News)

https://ministack.org/

# Summary

MiniStack is a free, open-source alternative to LocalStack that emulates 33 AWS services locally without requiring accounts, licenses, or telemetry. It runs in Docker with minimal resource requirements (~30MB RAM idle, 150MB image size, ~2s startup) and provides real infrastructure for key services—RDS spins up actual Postgres/MySQL containers, ElastiCache runs real Redis, and ECS launches real Docker containers. The tool is fully compatible with AWS CLI, boto3, Terraform, and other standard AWS tools.

Unlike LocalStack, which moved core services behind a paid tier, MiniStack remains completely free under an MIT license and includes advanced features like Lambda execution, IAM, EventBridge, Step Functions, API Gateway, Cognito, EC2/VPC, and more. It outperforms LocalStack on startup time, memory usage, and image size while delivering the same developer experience for local development and CI/CD pipelines.
