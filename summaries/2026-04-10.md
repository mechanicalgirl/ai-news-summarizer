### Native Instant Space Switching on macOS - (Hacker News)

https://arhan.sh/blog/native-instant-space-switching-on-macos/

# Summary

The author discusses the frustration of macOS's slow space-switching animation and evaluates various solutions. Common recommendations like enabling "Reduce motion" or using the yabai tiling window manager have significant drawbacks—the former merely replaces one animation with another and affects web browsing, while the latter requires disabling System Integrity Protection and forces adoption of a specific window manager. Third-party alternatives like FlashSpace and AeroSpace feel unnecessarily complex.

The author's recommended solution is **InstantSpaceSwitcher**, a lightweight menu bar application that simulates high-velocity trackpad swipes to achieve instant space switching without security compromises or workflow disruptions. It also offers command-line access and the ability to jump directly to specific spaces. The author provides installation instructions and encourages readers to star the relatively unknown GitHub repository to increase its visibility and credibility.

### I still prefer MCP over skills - (Hacker News)

https://david.coffee/i-still-prefer-mcp-over-skills/

# Summary

The author argues that Model Context Protocol (MCP) remains superior to "Skills" for giving LLMs access to services, despite recent industry momentum toward Skills as a universal standard. While Skills excel at pure knowledge transfer (teaching LLMs how to use existing tools or organizational practices), they fall short for actual service integration because most require CLI installation—which doesn't work in web-based LLM environments like standard Claude or ChatGPT. MCPs, by contrast, provide cleaner architecture through remote APIs that require zero local installation, handle authentication gracefully via OAuth, enable seamless updates, and work across any device.

The author proposes a clear division of labor: MCPs should serve as "connectors" for interfacing with external services, while Skills should function as "manuals" for teaching knowledge and workflows. The ideal approach combines both—using remote MCPs for actual service access while supplementing them with Skills that document gotchas, conventions, and best practices. This hybrid model avoids the deployment, secret management, and fragmentation problems Skills currently introduce when forced to handle service integrations.

### Charcuterie – Visual similarity Unicode explorer - (Hacker News)

https://charcuterie.elastiq.ch/

# Summary

Charcuterie is an interactive web-based tool that allows users to visually explore Unicode characters. It enables browsing of the character set, discovering related glyphs, and learning about the scripts, symbols, and shapes within the Unicode standard. The tool uses SigLIP 2 embeddings to identify visually similar characters and compare them in vector space.

The project is actively maintained by David Aerne and runs entirely in the browser, requiring no server-side processing. The creator welcomes feedback and donations to support ongoing development.

### Principles of Mechanical Sympathy - (Hacker News)

https://martinfowler.com/articles/mechanical-sympathy-principles.html

# Summary

"Principles of Mechanical Sympathy" explains how software can be optimized by understanding and working with underlying hardware capabilities. The article introduces four key principles: predictable memory access (leveraging CPU cache hierarchies through sequential rather than random data access), awareness of cache lines and false sharing (preventing multiple threads from inefficiently competing over the same cache line), the single-writer principle (dedicating one thread to all writes to a resource to eliminate mutex overhead), and natural batching (grouping independent requests efficiently without arbitrary timeouts or fixed intervals).

These principles, borrowed from high-frequency trading engineer Martin Thompson's work, can dramatically improve system performance across applications—from AI inference servers to ETL pipelines. By understanding how CPUs organize memory and making software decisions that align with hardware behavior, developers can create surprisingly performant systems without requiring complex optimizations.

### PicoZ80 – Drop-In Z80 Replacement - (Hacker News)

https://eaw.app/picoz80/

# picoZ80 Summary

The picoZ80 is a custom PCB replacement for the Z80 processor in legacy computers that drops directly into a Z80 DIP-40 socket. It uses an RP2350B microcontroller with dual-core 150MHz Cortex-M33 processors to provide cycle-accurate Z80 bus emulation while adding modern capabilities like WiFi connectivity, SD card storage, virtual devices, and configurable machine personas. The board maintains exact Z80 bus timing through dedicated PIO state machines while leveraging the RP2350's second core and abundant memory (8MB external PSRAM, 16MB Flash) to layer advanced features like RAM/ROM banking, disk emulation, and a browser-based management interface.

The picoZ80 has been successfully demonstrated on Sharp MZ computers and is designed to support various Z80-based systems through JSON-driven configuration files that require no recompilation. Key features include floppy and QuickDisk emulation, WiFi and web management via an onboard ESP32 co-processor, dual firmware partitions for safe updates, and a compact six-layer PCB design that fits within the physical constraints of a DIP-40 package while integrating five subsystems including power management and USB connectivity.
