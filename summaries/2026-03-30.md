### ChatGPT won't let you type until Cloudflare reads your React state - (Hacker News)

https://www.buchodi.com/chatgpt-wont-let-you-type-until-cloudflare-reads-your-react-state-i-decrypted-the-program-that-does-it/

# Summary

The article reveals that Cloudflare's Turnstile bot detection system running on ChatGPT uses encrypted bytecode that can be decrypted by analyzing network traffic. The author decrypted 377 instances of the program and found it checks 55 properties across three layers: browser fingerprinting (GPU, screen, fonts), Cloudflare network data (IP geolocation), and ChatGPT's React application state (internal framework objects). Notably, the system verifies not just that a real browser is running, but that the specific ChatGPT React application has fully loaded and hydrated—defeating bots that spoof browser APIs without executing actual JavaScript.

The encryption protecting these programs uses XOR with a key that is embedded in the same bytecode payload sent by the server, making it trivial to decrypt with the HTTP request and response alone. Beyond fingerprinting, the system also runs behavioral analysis monitoring keystroke timing and mouse patterns, plus a proof-of-work challenge. While the obfuscation prevents casual inspection, it provides no real cryptographic security, as the privacy boundary depends entirely on policy rather than actual encryption strength.

### VHDL's Crown Jewel - (Hacker News)

https://www.sigasi.com/opinion/jan/vhdls-crown-jewel/

# Summary

This article explains VHDL's "delta cycle algorithm," which is the language's key feature for preserving determinism in concurrent hardware design. VHDL separates signal value updates and process evaluations into distinct phases within each delta cycle, ensuring that processes always see consistent signal values regardless of execution order. This architectural separation guarantees deterministic results even though the internal order of operations is undefined.

In contrast, Verilog lacks this separation and cannot guarantee determinism. While Verilog's nonblocking assignments attempt to address this by delaying updates to future cycles, they only solve the problem for synchronous designs—not for more general cases like testbenches and high-level models. The author argues that VHDL's approach is a simple, zero-cost solution to an important problem that Verilog fundamentally missed in its design.

### 15 Years of Forking - (Hacker News)

https://www.waterfox.com/blog/15-years-of-forking/

# Summary

Waterfox, a Firefox fork created 15 years ago by then-16-year-old Alex Kontos to provide a 64-bit browser build, has grown into an independent browser with approximately 1 million monthly active users and over 25 million lifetime downloads. Starting as a hobby project on SourceForge, Waterfox evolved through various ownership phases, including a stint under System1, before returning to independence under BrowserWorks. The browser has maintained its focus on privacy and user choice throughout its history.

In 2026, Waterfox is introducing a native content blocker built on Brave's adblock library while continuing to prioritize user privacy and avoiding AI integration. Despite economic challenges from reduced search partnership revenue, Kontos remains committed to the project, sustained by community support and a user base that values a browser designed to respect privacy rather than serve corporate interests. Looking forward, the browser aims to expand distribution across different platforms while maintaining its core mission of providing fast, private, and user-friendly web browsing.

### Coding Agents Could Make Free Software Matter Again - (Hacker News)

https://www.gjlondon.com/blog/ai-agents-could-make-free-software-matter-again/

# Summary

The article argues that AI coding agents could revitalize the concept of software freedom by making source code access practically valuable rather than merely symbolic. Historically, Richard Stallman's "free software" movement—emphasizing user rights to run, study, modify, and share software—lost relevance when SaaS made it irrelevant whether users could access source code, since software ran on vendors' servers rather than users' machines. However, if AI agents can understand and modify codebases on behalf of non-programmer users, source code access becomes a practical capability for everyone, not just developers.

The article traces how "free software" was rebranded as "open source" in 1998 to appeal to corporations, stripping away the ethical dimension while keeping code-sharing practices. SaaS further undermined software freedom by exploiting licensing loopholes—vendors could use free software internally without sharing modifications. The author illustrates this shift through personal experience: attempting to customize a closed SaaS task-management tool revealed how proprietary software locks users into inflexible workflows. With AI agents as intermediaries, the freedom to modify software transforms from an abstract principle into a tangible user benefit, potentially making free software matter again in a way the SaaS era made seem impossible.

### Philly courts will ban all smart eyeglasses starting next week - (Hacker News)

https://www.inquirer.com/news/philadelphia/smart-glasses-ai-meta-courts-20260326.html

# Summary

The First Judicial District of Pennsylvania has announced a ban on all smart eyeglasses starting Monday, March 30, 2026, in all courthouses and judicial buildings. The ban applies to any eyewear with video and audio recording capabilities, even prescription glasses, because such devices are difficult to detect and pose a risk of witness and juror intimidation through unauthorized recording threats. Violators could be barred entry, removed from buildings, or arrested for criminal contempt, though judges may grant prior written exceptions.

Philadelphia joins an early wave of court systems implementing smart eyewear bans, including those in Hawaii, Wisconsin, and North Carolina. The move comes as affordable smart glasses—such as Ray-Ban and Oakley models with Meta AI integration selling for under $500—have become increasingly popular, with 7 million pairs sold in 2025. The ban reflects growing judicial concerns about recording devices that are harder to monitor than traditional phones and laptops, which remain allowed but must be powered off.

### C++26 is done: ISO C++ standards meeting Trip Report - (Hacker News)

https://herbsutter.com/2026/03/29/c26-is-done-trip-report-march-2026-iso-c-standards-meeting-london-croydon-uk/

# Summary

C++26 has been officially completed by the ISO C++ committee at their March 2026 meeting in London, with approximately 210 attendees representing 24 nations. The standard is now undergoing final editorial work before international approval. The release is considered the most significant since C++11, featuring four major additions: reflection (enabling C++ to describe and generate its own code), memory safety improvements (eliminating undefined behavior for uninitialized variables and providing hardened standard library protections), contracts (preconditions, postconditions, and assertions), and std::execution (a unified framework for safe concurrency and parallelism).

The committee expects rapid industry adoption of C++26 due to extraordinarily high user demand for these features—particularly reflection and safety improvements—and because major compiler implementations (GCC and Clang) already have most features partially or fully implemented. Looking forward, C++29 development has begun with a focus on further enhancing memory safety and developing safety profiles to help developers write more secure code.

### Douglas Lenat's Automated Mathematician Source Code - (Hacker News)

https://github.com/white-flame/am

# Summary

This GitHub repository contains Douglas Lenat's AM (Automated Mathematician), a landmark artificial intelligence program from 1977 that was recovered from the SAIL archives. The code was originally written in Interlisp and ran on a PDP-10 computer with 256k of memory, and is now being preserved in a public repository along with Lenat's accompanying PhD thesis.

The repository serves as a historical archive of early AI research, with the software placed in the public domain since it was originally funded by the U.S. government through ARPA. The maintainer notes that additional documentation and demonstration logs from the original SAILDART archive may be added, and the project is connected to Lenat's later work on the EURISKO project.
