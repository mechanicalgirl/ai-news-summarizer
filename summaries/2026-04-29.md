### Welcome to the “find out” stage of AI - (Stack Overflow Blog)

https://stackoverflow.blog/2026/04/27/welcome-to-the-find-out-stage-of-ai/

# Summary

The AI industry has transitioned from the experimental "wonder and surprise" phase into a practical "find out" stage where companies must demonstrate real business value. After going through renewal cycles with customers, organizations are moving beyond simple chatbots to deploy autonomous agents that perform complex, multi-step tasks—but this shift has exposed critical challenges around trust and reliability. Since mistakes in high-stakes sectors like healthcare and law can have serious consequences, the focus has shifted from improving model performance to ensuring these systems are trustworthy, hallucination-free, auditable, and properly permissioned.

Alongside reliability concerns, companies are grappling with skyrocketing token costs as agents consume more computational resources through expanded context windows, multi-step reasoning, and agent swarms. While no one has solved the profitability puzzle yet—even major players like OpenAI and Anthropic don't expect profits until 2028-2030—organizations are frantically seeking ways to optimize spending and monetization models. There's also growing anxiety about AI's broader social impacts, particularly regarding bias, critical thinking, and well-being, though these concerns remain largely informal discussions as the technology is too new to fully understand its long-term effects.

### Bugs Rust won't catch - (Hacker News)

https://corrode.dev/blog/bugs-rust-wont-catch/

# Summary

This article examines 44 CVEs discovered in uutils, a Rust reimplementation of GNU coreutils, to identify security bugs that Rust's type system and standard tools cannot prevent. The author highlights four major bug categories: Time-of-check-time-of-use (TOCTOU) vulnerabilities where attackers can swap paths between syscalls, permission races where files briefly exist with insecure permissions before being fixed, path comparison bugs where different string representations of the same path bypass security checks, and unsafe UTF-8 conversions that corrupt binary data.

The key lessons are practical rules for systems programming in Rust: anchor operations on file descriptors rather than paths, set permissions at file creation time rather than after, canonicalize paths before comparing them, stay in bytes at Unix boundaries instead of converting to UTF-8 strings, and treat every panic as a potential denial-of-service vulnerability by converting runtime errors into proper error handling. While Rust's safety guarantees prevent many bugs, these examples show that memory safety alone doesn't guarantee security in systems code.

### How ChatGPT serves ads - (Hacker News)

https://www.buchodi.com/how-chatgpt-serves-ads-heres-the-full-attribution-loop/

# Summary

OpenAI has implemented a sophisticated ad platform within ChatGPT that injects advertisements directly into conversations through a backend SSE stream. Ads are selected based on contextual relevance to the chat topic and displayed as structured ad units containing merchant information, creative assets, and encrypted tracking tokens. The system uses four distinct Fernet-encrypted tokens per ad for integrity verification and attribution purposes.

The attribution loop is completed through a tracking SDK called OAIQ that runs on merchant websites. When users click an ad, the browser opens the merchant site with tracking parameters including the `oppref` token, which the OAIQ SDK stores in a 30-day cookie and uses to report all subsequent user actions back to OpenAI's servers. This creates a closed-loop system where OpenAI can track impressions, clicks, and post-click conversions while maintaining encrypted security measures throughout the process.

### Before GitHub - (Hacker News)

https://lucumr.pocoo.org/2026/4/28/before-github/

# Summary

Armin Ronacher reflects on GitHub's rise and current decline, tracing the evolution of open source infrastructure from SourceForge and self-hosted solutions to GitHub's dominance as a centralized platform. He acknowledges GitHub's tremendous positive impact—making project creation and contribution accessible, providing discoverable archives of software commons, and normalizing open collaboration—while recognizing the irony that distributed version control systems led to standardization on one centralized service.

However, Ronacher argues that GitHub's current instability, unclear leadership, and perceived shift away from community priorities signal its slow decline. He notes that while GitHub enabled the "frictionless" micro-dependency ecosystem through npm integration and trusted publishing, it also created dependency on a single platform for trust and supply chain security. Recent high-profile projects like Ghostty moving away from GitHub suggest developers are increasingly seeking alternatives, marking a potential shift back toward distributed infrastructure and away from single-platform dominance.

### Show HN: Rocky – Rust SQL engine with branches, replay, column lineage - (Hacker News)

https://github.com/rocky-data/rocky

# Rocky Summary

Rocky is an open-source, Rust-based control plane for data warehouse pipelines that works alongside Databricks or Snowflake. It provides advanced data management features including named branches for risk-free experimentation, column-level lineage tracking, compile-time data contract enforcement, automatic schema drift detection, and per-model cost attribution. The tool is designed to catch data quality issues and breaking changes before they impact production.

The project is available as a CLI binary, Dagster integration, and VS Code extension, with comprehensive documentation and examples. Users can get started in 60 seconds using a local DuckDB playground, and the codebase is written primarily in Rust with Python and TypeScript components. Rocky is free and open-source under the Apache 2.0 license.

### Show HN: Auto-Architecture: Karpathy's Loop, pointed at a CPU - (Hacker News)

https://github.com/FeSens/auto-arch-tournament/blob/main/docs/auto-arch-tournament-blog-post.md

# Summary

This article describes an experiment using an autonomous research loop (inspired by Andrej Karpathy's autoresearch) to optimize CPU microarchitecture design. The researcher pointed an LLM agent at a 5-stage in-order RISC-V CPU core written in SystemVerilog, letting it propose and implement microarchitectural optimizations over ~10 hours. The agent tested 73 hypotheses across five categories, with 10 accepted improvements that increased performance by 92% over the baseline and 56% over the human-designed VexRiscv benchmark, while actually reducing chip size (LUT count by 40%).

The article's key insight is that the autonomous loop itself is not novel—the real value lies in the **verifier**, the unglamorous but essential infrastructure that gates changes. The verifier includes formal verification checks, path sandboxing to prevent the agent from modifying test harnesses, statistical validation through multi-seed place-and-route, and CRC validation to ensure correctness. Without these safeguards, the agent's 63 failed hypotheses would have corrupted the design or misled the system into accepting bad optimizations as improvements.

### Germany Overtakes US in Ammunition Production Capacity - (Hacker News)

https://www.newsweek.com/germany-overtakes-us-in-ammunition-production-capacity-11886409

# Summary

Germany has surpassed the United States in ammunition production capacity, according to Rheinmetall's CEO. The German defense company has quadrupled its medium-caliber ammunition output and increased artillery round production to 1.1 million annually, up from just 70,000. This surge reflects Europe's major rearmament campaign, driven by concerns about reduced U.S. military support under the Trump administration and the need to strengthen defenses following Russia's invasion of Ukraine.

European nations are investing heavily in military capabilities after Trump signaled the U.S. would prioritize the Indo-Pacific over Europe. NATO members pledged to spend 5% of GDP on defense by 2035, marking the most significant military investment increase in decades. Germany is leading this effort, aiming to build Europe's strongest conventional army by 2039. The high demand for ammunition, particularly 155mm artillery rounds depleted in Ukraine, has driven investment in new production facilities across Europe.
