### Apple Foundation Models - (Hacker News)

https://platform.claude.com/docs/en/cli-sdks-libraries/libraries/apple-foundation-models

# Summary

Claude for Foundation Models is a Swift package that integrates Claude AI into Apple's Foundation Models framework, allowing developers to use Claude as a server-side language model alongside Apple's on-device model. The package enables seamless switching between models using the same LanguageModelSession API and supports features like streaming, structured output, tool use, and image input. Requests go directly to the Claude API (billed to your Anthropic account) with Apple never seeing the data.

The package requires iOS/macOS/visionOS/watchOS 27 (beta) and can be easily installed via Swift Package Manager. Developers can choose from pre-defined Claude models or create custom ones, authenticate via API key during development or through a production proxy, and leverage server-side tools like web search and code execution. The package is currently in beta and targets the new Foundation Models server-side API.

### Show HN: Kage – Shadow any website to a single binary for offline viewing - (Hacker News)

https://github.com/tamnd/kage

# Kage Summary

**Kage** is an open-source tool that clones websites into offline-browsable folders with all JavaScript stripped out. It uses a headless Chrome browser to render each page, captures the final DOM state that users would see, then removes all scripts while preserving CSS, images, and fonts as local files. This creates a static, script-free copy that can be read offline without network calls or tracking.

The tool offers multiple output formats: a browsable folder, a compressed ZIM archive (compatible with the Kiwix ecosystem), or a self-contained executable. Key features include polite crawling with robots.txt support, selective scope cloning, lazy-load triggering via auto-scroll, and idempotent operations that can be resumed or refreshed. Installation is simple via `go install`, and the tool includes a local preview server for testing cloned sites before packing them.

# Summary
