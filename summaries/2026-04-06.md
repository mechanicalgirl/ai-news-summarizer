### SideX – A Tauri-based port of Visual Studio Code - (Hacker News)

https://github.com/Sidenai/sidex

# SideX: A Lightweight Tauri-Based Code Editor

SideX is an early-stage, open-source port of Visual Studio Code that replaces Electron with Tauri (a Rust backend combined with native webviews), resulting in a 96% smaller application footprint. It maintains VSCode's original architecture and features—including the Monaco editor, integrated terminal, file explorer, Git integration, and extension support—while aiming for significantly lower memory usage (200MB RAM goal). The project has ported over 5,600 TypeScript files from VSCode's source and eliminated all Electron dependencies.

Currently in active development, SideX has core features working (editor, syntax highlighting, file management, terminal, and theme support) but many advanced features remain unstable or incomplete, such as full extension host compatibility, debugging support, and multi-window functionality. The project was intentionally released early to invite community contributions, with particular needs in terminal stability, extensions compatibility, file system reliability, and cross-platform testing.

### Show HN: I built a tiny LLM to demystify how language models work - (Hacker News)

https://github.com/arman-bd/guppylm

# GuppyLM Summary

GuppyLM is a ~9M parameter language model trained to roleplay as a small fish named Guppy. The project demonstrates that training a functional LLM doesn't require massive resources—it can be done in ~5 minutes on a single GPU in a Colab notebook. The model uses a vanilla transformer architecture and is trained on 60K synthetic conversations across 60 topics, generating responses in short, lowercase sentences focused on fish-centric concerns like food, water temperature, and bubbles.

The project's primary goal is educational: to demystify language model development by showing all steps from data generation and tokenization to architecture design and inference. GuppyLM deliberately avoids complex optimizations (GQA, RoPE, etc.) to keep the code simple and understandable. The model is small enough to run in a browser and comes with pre-trained weights available on HuggingFace, allowing users to chat with Guppy immediately or train their own version following the provided notebooks.

### An open-source 240-antenna array to bounce signals off the Moon - (Hacker News)

https://moonrf.com/

# Summary

MoonRF is an open-source initiative democratizing Earth-Moon-Earth (EME) communication by developing affordable, software-defined phased array hardware that was traditionally only accessible to well-funded radio amateurs. Their modular approach begins with the QuadRF, a $49-99 SDR tile that can be scaled up into larger arrays like the Mini (18 tiles, $899-$1,499) and Moon (60 tiles, $2,499-$4,999), allowing users to experiment with moon-bounce communications, radio astronomy, and other RF applications.

The hardware, expected to ship in July 2026, operates in the C-band (4.9–6 GHz) and features flexible architecture for both standalone use and arraying. While requiring an Amateur Radio license to operate, these kits aim to bring space communication capabilities to the hobbyist community by providing the necessary transmit power and receive sensitivity for moon-bounce experiments without the massive antennas and expensive equipment previously required.

### Show HN: Real-time AI (audio/video in, voice out) on an M3 Pro with Gemma E2B - (Hacker News)

https://github.com/fikrikarim/parlor

# Parlor Summary

**Parlor** is an open-source project that enables real-time multimodal AI conversations (voice and vision) that run entirely on your local machine without requiring server infrastructure. It uses Google's Gemma 4 E2B model for speech and vision understanding, combined with Kokoro for text-to-speech, allowing users to have natural conversations with an AI by speaking and showing their camera.

The project was created to provide a sustainable, cost-free solution for language learning applications. It supports browser-based voice activity detection, allows interrupting the AI mid-sentence, and achieves impressive performance on consumer hardware (e.g., ~2.5-3 seconds end-to-end latency on an M3 Pro). The system is easy to set up, automatically downloads required models (~2.6 GB), and is available under an Apache 2.0 license, though the creator notes it's currently in research preview with potential rough edges.

### Signals, the push-pull based algorithm - (Hacker News)

https://willybrauner.com/journal/signal-the-push-pull-based-algorithm

# Summary

Signals are a modern implementation of Reactive Programming that automatically propagate changes through dependent values—similar to how cells update in a spreadsheet. The article explains that Signals use a "push-pull based algorithm" combining two approaches: the **push-based (eager) mechanism** where signals immediately notify subscribers when they change, and the **pull-based (lazy) mechanism** where computed values only re-evaluate when accessed after being invalidated by dependency changes.

The key innovation is automatic dependency tracking: unlike React's manual dependency arrays, computed values automatically discover which signals they depend on during execution. This is achieved through a global stack that tracks which computed function is currently running, allowing signals to register themselves as dependencies when accessed. The system uses cache invalidation (dirty flags) to optimize performance, ensuring computed values are recalculated only when necessary—making the reactive system both elegant and efficient.
