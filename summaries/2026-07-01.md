https://stackoverflow.blog/2026/06/30/why-intent-prediction-needs-more-than-an-llm/

# Summary

Frank Portman, CTO at Yobi, argues that Large Language Models (LLMs) are fundamentally ill-suited for intent prediction and behavioral forecasting because their inductive bias—predicting the next token in text sequences—doesn't translate well to decision-making under uncertainty. While LLMs excel at synthesizing information and generating text, the skills required for accurate behavior prediction are substantially different and require models specifically designed for that task.

To address this gap, Yobi has developed what Portman calls a "foundation model of behavior" using transformers and graph neural networks rather than chat-style LLMs. This approach trains on proprietary, sensitive behavioral data (not just text) to create broadly predictive representations that can generalize to new products and scenarios. Yobi positions itself as a behavioral AI company operating in ad tech and marketing, but with aspirations to expand into any domain requiring personalization decisions and recommendations.

### The first early human eggs from stem cells - (Hacker News)

https://www.conception.bio/science-and-updates/the-first-early-human-eggs-from-stem-cells

# Summary

Conception has achieved a major scientific breakthrough by creating the first early human egg cells (primary oocytes) derived from stem cells. The process begins with a simple blood draw, converting blood cells into induced pluripotent stem cells (iPSCs), which are then guided to become miniature human ovaries containing early eggs. While the eggs still need to mature to full development, this represents a significant advance in "in vitro gametogenesis" (IVG), a technology previously demonstrated only in mice.

This technology has transformative potential for fertility and reproduction, as it could theoretically allow unlimited egg production from a single blood sample, eliminating the need for hormone injections and surgical procedures required in traditional IVF. Conception's rigorous approach focuses on recreating the natural developmental sequence by building complete mini-ovaries with both egg cells and supporting cells, benchmarking their development against human biology at every stage. Key achievements include demonstrating proper ovarian structure formation, successful progression through meiosis (the critical cell division that produces eggs with single chromosome sets), and the creation of fully stem cell-derived follicles—structures that house and nurture developing eggs.

### Google copybara: moving code between repositories - (Hacker News)

https://github.com/google/copybara

# Copybara Summary

Copybara is Google's open-source tool designed to transform and move code between repositories while maintaining a single source of truth. It addresses the common need to keep multiple repositories (such as confidential and public versions) synchronized by allowing selective code transformations and transfers between them. The tool supports Git repositories natively and experimental Mercurial support, with an extensible architecture for custom origins and destinations.

Key features include its stateless design, which stores state in destination repository commit messages, enabling multiple users to achieve consistent results. Common use cases include syncing code between public and private repositories, importing sections of code, and handling contributions across non-authoritative repositories. The project is built with Java/Bazel and can be installed via pre-built releases, compiled from source, or integrated into Bazel projects.

### Matrix Orthogonalization Improves Memory in Recurrent Models - (Hacker News)

https://ayushtambde.com/blog/matrix-orthogonalization-improves-memory-in-recurrent-models/

# Summary

This article presents a technique for improving memory in recurrent neural networks (RNNs) by orthogonalizing the memory matrix during read operations. While Transformers excel at associative recall through attention mechanisms, they're computationally expensive for certain applications like long-horizon reinforcement learning. The authors tested whether applying matrix orthogonalization—inspired by the Muon optimizer's success in preventing weaker memories from being overshadowed—could enhance mLSTM performance on noisy associative recall (NAR) tasks, where models must retrieve correct values despite distracting noise.

The experiments demonstrated consistent improvements across various vocabulary sizes and sequence lengths, with especially dramatic gains in harder tasks (vocabulary 96): orthogonalized mLSTMs increased success rates from 4/24 to 14-16/24 solved seeds. However, the authors acknowledge limitations, noting that results are from small models tested on synthetic NAR benchmarks and that it remains unclear whether these improvements transfer to real-world applications at larger scales.
